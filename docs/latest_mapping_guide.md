## Linux OS 점검/조치 지원 매트릭스

아래 표는 `linux_vuln_check.sh`의 최신 `U-01~U-67` 점검 항목과 `linux_vuln_fix.sh` 조치 레이어의 현재 지원 상태를 정리한 것입니다.

- **직접 자동 조치**: 스크립트가 파일 권한, 서비스 중지/비활성화, 설정값 보강 등을 최신 점검 기준에 맞춰 직접 수행합니다.
- **레거시 매핑**: 기존 조치 함수를 최신 코드에 연결한 상태이며, 후속 항목별 의미 검증이 필요할 수 있습니다.
- **부분 자동/수동 확인**: 안전한 범위는 자동 조치하지만 정책 판단 또는 환경별 설정은 수동 확인이 필요합니다.
- **수동 조치**: 업무 영향 또는 기관별 정책 판단이 필요하여 스크립트가 안내만 제공합니다.
- **N/A 처리**: 점검 결과가 `PASS` 또는 `N/A`이면 조치 스크립트는 `SKIPPED`로 처리합니다.
